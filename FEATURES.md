# 기능 명세 (FEATURES)

## 1. SSH 터미널 핵심 기능

### 1.1 서버 접속 관리 (Putty 수준 이상)

| 기능 | 설명 |
|------|------|
| 서버 프로필 저장 | 이름, IP, 포트, 계정, 인증방식 저장 |
| 폴더/그룹 분류 | 서버를 그룹으로 묶어 관리 |
| 즐겨찾기 | 자주 쓰는 서버 상단 고정 |
| 최근 접속 | 최근 접속 서버 목록 |
| 검색 | 서버명/IP로 빠른 검색 |
| 원클릭 접속 | 탭 선택 → 즉시 SSH 연결 |

### 1.2 인증 방식

```
지원 인증
├── Password 인증
├── Public Key 인증 (.pem, .ppk, .pub)
├── SSH Agent 연동
└── 키 파일 앱 내 보관 (암호화)
```

### 1.3 터미널 기능

| 기능 | 설명 |
|------|------|
| 멀티탭 | 여러 서버 동시 접속 탭 |
| 분할화면 | 좌우/상하 화면 분할 |
| 폰트/색상 | 터미널 테마 설정 |
| 폰트 크기 | 손가락으로 핀치 조절 (모바일) |
| 스크롤백 | 버퍼 크기 설정 |
| 복사/붙여넣기 | 터미널 텍스트 선택/복사 |
| 키보드 단축키 | Ctrl+C, Ctrl+D 등 |
| 세션 유지 | 백그라운드 keep-alive |

### 1.4 모바일 전용 기능

```
안드로이드/iOS
├── 특수키 툴바 (Ctrl, Alt, Esc, Tab, 화살표)
├── 핀치 줌 (폰트 크기 조절)
├── 스와이프로 탭 전환
├── 햅틱 피드백
└── 세로/가로 모드 자동 전환
```

## 2. CPM 통합 기능

### 2.1 CPM 대시보드

```
앱 하단 탭 or 사이드패널
├── 현재 프로젝트 토큰 사용량
├── 최근 프롬프트 목록
├── 프로젝트 목록
└── 서비스 포트 현황
```

### 2.2 프롬프트 관리

| 기능 | 설명 |
|------|------|
| 히스토리 조회 | 프로젝트별 프롬프트 목록 |
| 빠른 재사용 | 과거 프롬프트 탭 → SSH로 전송 |
| 템플릿 | 자주 쓰는 프롬프트 템플릿 저장 |
| 검색 | 키워드로 프롬프트 검색 |
| 상태 표시 | success / fail / pending |
| 태그 필터 | bug, feature, refactor 등 |

### 2.3 프롬프트 빠른 전송

```
터미널 화면 내 프롬프트 버튼
→ 템플릿 선택 or 직접 입력
→ SSH 채널로 Claude Code에 전송
→ CPM hook이 자동으로 DB 저장
→ 실시간 응답 터미널에 표시
```

### 2.4 실시간 WebSocket 연동

```
CPM ws:// 연결
→ 프롬프트 저장 이벤트 수신
→ 앱 대시보드 자동 업데이트
→ 토큰 사용량 실시간 반영
```

## 3. 프로젝트 연동 기능

### 3.1 프로젝트별 SSH 세팅

```
CPM 프로젝트 1개 = SSH 세팅 1세트
├── 접속 서버 (SSH 프로필)
├── 초기 디렉토리 (cd /path/to/project)
├── 초기 실행 명령어 (자동 실행)
└── CPM 프로젝트 ID 연결
```

### 3.2 원클릭 프로젝트 접속

```
예시:
[ai100 프로젝트] 탭 선택
→ SSH 192.168.219.100 자동 접속
→ cd /home/ai100 자동 실행
→ CPM 대시보드에 ai100 프로젝트 로드
→ 바로 작업 시작 가능
```

### 3.3 서비스 포트 모니터링

```
CPM /api/services/ 데이터
→ 프로젝트별 실행 중인 서비스 목록
→ 포트 상태 (열림/닫힘)
→ 탭으로 브라우저 열기 (웹서비스인 경우)
```

## 4. 설정 기능

### 4.1 CPM 서버 설정

```
CPM 서버 주소: http://192.168.219.x:9200
→ 자동 연결 테스트
→ 연결 실패 시 오프라인 모드
```

### 4.2 터미널 외관 설정

| 설정 | 옵션 |
|------|------|
| 테마 | Dark / Light / Solarized / Monokai |
| 폰트 | JetBrains Mono / Fira Code / Hack |
| 폰트 크기 | 10 ~ 20px |
| 커서 | Block / Underline / Bar |
| 배경 투명도 | 0 ~ 100% |

### 4.3 SSH 기본 설정

```
기본 포트: 22
Keep-alive 간격: 30초
연결 타임아웃: 10초
스크롤백 버퍼: 10000줄
인코딩: UTF-8
```
