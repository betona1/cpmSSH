## 기술 부채 및 주의사항

```
dartssh2 주의:
- 일부 구형 서버 SSH 알고리즘 미지원 가능
- RSA 키는 OpenSSH 형식 권장

xterm 주의:
- 복잡한 TUI 앱 (vim, htop) 렌더링 확인 필요
- 모바일 IME 한글 입력 처리 주의

CPM API:
- CSRF 토큰 처리 필요 (POST 요청 시)
- 버전 업데이트 시 API 변경 확인
```
