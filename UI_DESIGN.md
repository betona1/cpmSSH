## 주요 화면 목록

### 1. 서버 목록 화면 (메인)
```
기능:
- 서버 카드 목록 (이름, IP, 마지막 접속)
- 그룹별 폴더 구조
- 상단 검색바
- 우하단 FAB → 서버 추가
- 스와이프 → 편집/삭제
- 탭 → 즉시 SSH 접속
```

### 2. 서버 추가/편집 화면
```
입력 필드:
- 표시 이름 (필수)
- 호스트 IP or 도메인 (필수)
- 포트 (기본 22)
- 사용자명 (필수)
- 인증방식: Password / Private Key
  → Password: 비밀번호 입력
  → Private Key: 파일 선택 or 붙여넣기
- 초기 디렉토리 (선택)
- 초기 명령어 (선택)
- CPM 프로젝트 연결 (선택)
- 그룹 선택 (선택)
```

### 3. SSH 터미널 화면
```
구성:
- 상단: 탭바 (멀티세션)
- 중앙: xterm 터미널
- 하단 (모바일): 특수키 툴바
- 우상단: 프롬프트 전송 버튼 (💬)

액션:
- 텍스트 길게 누르기 → 복사/붙여넣기
- 두 손가락 핀치 → 폰트 크기
- 스와이프 → 탭 전환
```

### 4. CPM 대시보드 화면
```
섹션:
- 상단 카드: 총 프롬프트 / 프로젝트 수 / 토큰
- 프로젝트 목록 (원클릭 접속)
- 최근 프롬프트 목록
- 서비스 포트 현황
```

### 5. 프롬프트 히스토리 화면
```
기능:
- 프로젝트 필터 드롭다운
- 태그 필터 칩 (bug/feature/refactor...)
- 상태 필터 (success/fail/pending)
- 검색바
- 프롬프트 카드 → 탭으로 SSH에 전송
```

### 6. 프롬프트 전송 바텀시트
```
터미널 화면에서 💬 버튼 클릭 시:
┌─────────────────────────┐
│ 프롬프트 전송            │
├─────────────────────────┤
│ [템플릿 선택 ▼]          │
│                         │
│ ┌─────────────────────┐ │
│ │ 직접 입력...        │ │
│ └─────────────────────┘ │
│                         │
│ 태그: [bug][feature][+] │
│                         │
│      [취소]  [전송]      │
└─────────────────────────┘
```
